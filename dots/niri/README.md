# Niri Config

There's a few dependencies to get somethings working:

- If using gentoo don't forget the `screencast` USE flag
- XDG-desktop-portals are also required for screensharing and policykit integrations too (I think)

- Kanshi is used for handling display setup when using laptop docked

Tool Dependencies:
- brightnessctl
- ghostty
- fuzzel
- swaylock
- xwayland-satellite
- xdg-desktop-portal-gtk
- xdg-desktop-portal-gnome
